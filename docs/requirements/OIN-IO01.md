# OIN-IO01

## Purpose

Industrial digital input/output module.

---

## Functional requirements

FR-001

The module shall read one digital input.

FR-002

The module shall control one digital output.

FR-003

The module shall periodically transmit its status via UART.

---

## Non-functional requirements

NFR-001

Firmware shall be written in C.

NFR-002

The code shall be modular.

NFR-003

No dynamic memory allocation.

NFR-004

The module shall support future CAN integration.
